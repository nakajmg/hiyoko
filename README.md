# hiyoko

Markdown editor app for [esa.io](https://esa.io/).

## build

```sh
$ npm i
```

```sh
npm i -g electron-packager
```

```sh
cd ..
```

```sh
$ electron-packager ./hiyoko hiyoko --platform=darwin --arch=x64 --version=0.33.4
```

(\\( ⁰⊖⁰)/)

## Usage

* 設定アイコンをクリック
* Tokenとチーム名を設定
* Activaion
* Activaionに成功したら記事の投稿・更新・削除（ローカル＆リモートから）ができます。
